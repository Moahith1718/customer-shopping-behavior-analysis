# 📊 Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using **Python (Pandas), MySQL, and Power BI** to uncover purchasing patterns, customer preferences, and business insights.

---

## 🛠 Tools & Technologies
- Python (Pandas)
- MySQL
- Power BI
- Jupyter Notebook

---

## 📂 Dataset
- Customer Shopping Behavior Dataset
- Total Customers: 3900
- Features: Age, Gender, Category, Purchase Amount, Location, Subscription Status, Review Rating, Shipping Type, etc.
